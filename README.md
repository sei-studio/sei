Summon any character into your game. An omni-game AI player mod.

Not a chatbot, a player. Your favorite character joins your world and plays alongside you — building, fighting, remembering. The story between you only happens once.

Download at [sei.gg](https://sei.gg).

## Shipped

- [x] Desktop launcher (macOS, Windows)
- [x] Minecraft (vanilla + CurseForge instances, auto Fabric + skin setup)
- [x] Persona authoring — write a character in plain English, the launcher expands them
- [x] Per-character persistent memory
- [x] Custom skins (PNG upload or Mojang username)
- [x] In-app update notifications

## Roadmap

- [ ] Online character sharing + cloud-hosted brain (no API key needed)
- [ ] In-game vision — characters that actually see what's on screen
- [ ] Modded Minecraft adapters (Pixelmon, Create, Better MC)
- [ ] Adapters for other games (Skyrim next)
