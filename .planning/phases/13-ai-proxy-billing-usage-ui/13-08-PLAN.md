Test cases:
1. Insufficient balance → preDeduct returns {status:'insufficient'} → forward returns 402 with body `{ error: 'payment_required' }`. No upstream fetch. No header injection.
2. Anthropic 429 (org tier hit) → upstream.status===429 → settleAsRefunded(reservationId) is called → forward returns 503 with body `{ error: 'service_at_capacity', retry_after_seconds: 30 }` and DOES NOT propagate the 429 to the client (D-44).
3. Anthropic 200 non-streaming (body.stream !== true) → JSON pass-through; settle() called with parsed usage; X-Sei-Remaining-Pct set on response from POST-settle remainingPct.
4. Anthropic 200 streaming (body.stream === true) → X-Sei-Remaining-Pct set BEFORE streamSSE opens, from POST-reservation balance. SSE bytes piped through verbatim. On stream close, settle() called with merged {input from message_start, output from message_delta}. If no message_start or no message_delta seen → settleAtReservation (safer default per PATTERNS pitfall #5).
5. Client abort (c.req.raw.signal triggers abort) → proxy's AbortController on upstream fetch fires → upstream socket closed → settleAtReservation called (we don't know real usage, must settle to avoid pinning).
6. Upstream timeout (AbortSignal.timeout(120_000)) → caught, settleAsRefunded, returns 504 `{ error: 'upstream_timeout' }`.
7. anthropic-version header is exactly '2023-06-01' on upstream call.
8. Request body forwarded VERBATIM (no field stripping; cache_control markers untouched per CROSS-FILE CONVENTION 11).

Test harness uses Hono's `app.request(...)` for the route, vi.mock'd fetch returning a ReadableStream of SSE bytes for the streaming test.

```ts
import type { Context } from 'hono';
import { streamSSE } from 'hono/streaming';
import type { PreDeductResult } from '../ledger/preDeduct.js';
import type { Usage } from './pricing.js';

const ANTHROPIC_URL = 'https://api.anthropic.com/v1/messages';
const UPSTREAM_TIMEOUT_MS = 120_000;
const SERVICE_AT_CAPACITY_RETRY_SECONDS = 30;

export type ForwardDeps = {
  preDeduct: (userId: string, estInput: number, maxOutput: number, model: string) => Promise<PreDeductResult>;
  settle: (resId: string, usage: Usage, model: string, callId: string | null) => Promise<void>;
  settleAsRefunded: (resId: string) => Promise<void>;
  settleAtReservation: (resId: string, callId: string | null) => Promise<void>;
  remainingPct: (userId: string) => Promise<number>;
  estimateInputTokens: (body: unknown) => Promise<number>;
  fetchImpl: typeof fetch;
  anthropicApiKey: string;
};

export async function forwardToAnthropic(c: Context<{ Variables: { userId: string } }>, deps: ForwardDeps): Promise<Response> {
  const userId = c.get('userId');

  // Read raw body text ONCE — forwarded verbatim upstream to preserve byte-for-byte
  // ordering of cache_control markers (T-13-08-07 / Pitfall: Anthropic prompt cache
  // hashes the prefix as raw JSON; any re-serialization with key reordering or
  // whitespace changes invalidates org-wide cache hits).
  const rawBody = await c.req.text();

  // Parse a LOCAL COPY ONLY for header peeks (model / max_tokens / stream).
  // Never re-stringify and never forward this parsed object — `rawBody` is the
  // upstream payload.
  let parsed: { model?: string; max_tokens?: number; stream?: boolean };
  try {
    parsed = JSON.parse(rawBody);
  } catch {
    return c.json({ error: 'invalid_json_body' }, 400);
  }
  const model = parsed.model ?? 'claude-haiku-4-5';
  const maxOutput = parsed.max_tokens ?? 1024;

  // 1. Estimate input (pass the parsed object — estimator doesn't need byte fidelity)
  const estInput = await deps.estimateInputTokens(parsed);

  // 2. Reserve (atomic; FOR UPDATE row lock in 13-01 RPC)
  const reservation = await deps.preDeduct(userId, estInput, maxOutput, model);
  if (reservation.status === 'insufficient') {
    return c.json({ error: 'payment_required' }, 402);
  }

  // 3. Compute X-Sei-Remaining-Pct from POST-reservation balance.
  // CRITICAL (RESEARCH Pitfall 1): must set BEFORE streamSSE opens.
  const pct = await deps.remainingPct(userId);

  // 4. Upstream fetch with timeout + cancellation propagation
  const upstreamController = new AbortController();
  const clientSignal = c.req.raw.signal;
  if (clientSignal) clientSignal.addEventListener('abort', () => upstreamController.abort(), { once: true });
  const timeoutId = setTimeout(() => upstreamController.abort(), UPSTREAM_TIMEOUT_MS);

  let upstream: Response;
  try {
    upstream = await deps.fetchImpl(ANTHROPIC_URL, {
      method: 'POST',
      headers: {
        'x-api-key': deps.anthropicApiKey,
        'anthropic-version': '2023-06-01',
        'content-type': 'application/json',
      },
      // VERBATIM forward — see T-13-08-07. Do NOT re-stringify `parsed`.
      body: rawBody,
      signal: upstreamController.signal,
    });
  } catch (err: unknown) {
    clearTimeout(timeoutId);
    await deps.settleAsRefunded(reservation.reservationId);
    if ((err as { name?: string })?.name === 'AbortError') {
      return c.json({ error: 'upstream_timeout' }, 504);
    }
    return c.json({ error: 'upstream_error' }, 502);
  }

  // 5. Anthropic 429 → translate to 503 service_at_capacity (D-44)
  if (upstream.status === 429) {
    clearTimeout(timeoutId);
    await deps.settleAsRefunded(reservation.reservationId);
    return c.json({ error: 'service_at_capacity', retry_after_seconds: SERVICE_AT_CAPACITY_RETRY_SECONDS }, 503);
  }

  // 6. Set X-Sei-Remaining-Pct header (before any stream write)
  c.header('X-Sei-Remaining-Pct', String(pct));

  // 7. Non-streaming shortcut
  if (!parsed.stream) {
    clearTimeout(timeoutId);
    const respJson = await upstream.json() as { usage?: Usage; id?: string };
    if (respJson.usage) {
      await deps.settle(reservation.reservationId, respJson.usage, model, respJson.id ?? null);
    } else {
      await deps.settleAtReservation(reservation.reservationId, respJson.id ?? null);
    }
    return c.json(respJson as unknown as Record<string, unknown>, upstream.status as 200);
  }

  // 8. Streaming pass-through
  return streamSSE(c, async (stream) => {
    let inputUsage: Usage | null = null;
    let outputUsage: { output_tokens: number } | null = null;
    let messageId: string | null = null;

    const reader = upstream.body!.getReader();
    const decoder = new TextDecoder();
    let buf = '';
    try {
      while (true) {
        const { done, value } = await reader.read();
        if (done) break;
        buf += decoder.decode(value, { stream: true });
        let idx: number;
        while ((idx = buf.indexOf('\n\n')) !== -1) {
          const raw = buf.slice(0, idx);
          buf = buf.slice(idx + 2);
          // Parse for usage
          if (raw.includes('event: message_start')) {
            const dataLine = raw.split('\n').find(l => l.startsWith('data: '));
            if (dataLine) {
              try {
                const parsed = JSON.parse(dataLine.slice(6));
                inputUsage = parsed.message?.usage as Usage;
                messageId = parsed.message?.id ?? null;
              } catch { /* malformed; ignore */ }
            }
          } else if (raw.includes('event: message_delta')) {
            const dataLine = raw.split('\n').find(l => l.startsWith('data: '));
            if (dataLine) {
              try {
                const parsed = JSON.parse(dataLine.slice(6));
                outputUsage = parsed.usage;
              } catch { /* ignore */ }
            }
          }
          await stream.write(raw + '\n\n');
        }
      }
    } finally {
      clearTimeout(timeoutId);
      if (inputUsage && outputUsage) {
        const merged: Usage = {
          input_tokens: inputUsage.input_tokens,
          cache_creation_input_tokens: inputUsage.cache_creation_input_tokens ?? 0,
          cache_read_input_tokens: inputUsage.cache_read_input_tokens ?? 0,
          output_tokens: outputUsage.output_tokens,
        };
        try { await deps.settle(reservation.reservationId, merged, model, messageId); }
        catch (err) { console.error('settle failed', err); }
      } else {
        // PATTERNS pitfall #5: no message_delta → settle at full reservation, no refund
        try { await deps.settleAtReservation(reservation.reservationId, messageId); }
        catch (err) { console.error('settleAtReservation failed', err); }
      }
    }
  });
}
```

## Trust Boundaries

|----------|-------------|
| proxy → api.anthropic.com | x-api-key (org-level, in Fly secrets) |
| client request body → upstream | Forwarded verbatim; verifyJwt is the gate above |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-08-01 | Information Disclosure | x-api-key in upstream request | mitigate | Sourced from env (Fly secrets); never logged; only the upstream fetch sees it |
| T-13-08-02 | Tampering | Client forges X-Sei-Remaining-Pct in their request | n/a | proxy sets this header on RESPONSES; client-supplied headers ignored |
| T-13-08-03 | DoS | Long-running stream pins proxy machine | mitigate | AbortSignal.timeout(120_000) wall-clock; Fly.io concurrency soft_limit=20 / hard_limit=50 in fly.toml caps per-instance load |
| T-13-08-04 | DoS | Stream-interrupted reservation pins credits | mitigate | settleAtReservation in the finally block; settles to reservation amount so credits are not pinned forever |
| T-13-08-05 | Information Disclosure | Detailed upstream error reaches client | mitigate | Errors mapped to enumerated sentinels (payment_required / service_at_capacity / upstream_timeout / upstream_error / internal_error) |
| T-13-08-06 | Spoofing | Client passes invented model name | accept | computeMicroDollarCost falls back to default model pricing; reservation/settle math still consistent; Anthropic upstream rejects truly bogus models with 400, which we pass through |
